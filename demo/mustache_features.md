# Mustache Features Status

| Mustache Feature | Type (Core / Optional) | Used in Templates |
| :--- | :--- | :--- |
| **Variables (Escaped)** | Core | Yes |
| **Variables (Unescaped)** | Core | Yes |
| **Sections (Loop/Context/Condition)** | Core | Yes |
| **Inverted Sections** | Core | Yes |
| **Comments** | Core | Yes |
| **Partials** | Core | Yes (including recursive) |
| **Set Delimiter** | Core | Yes |
| **Lambdas** | Core | No (requires runtime code) |
| **Dynamic Partials** | Optional | No |
| **Template Inheritance / Blocks** | Optional | No |
