# Examples

Runnable examples live in this directory as standalone projects.

- [FastAPI](fastapi/) shows Fstache rendering into `HTMLResponse` and
  `StreamingResponse`.
- [Flask](flask/) shows Fstache rendering into regular and streamed WSGI
  `Response` objects.
- [Starlette](starlette/) shows Fstache rendering into `HTMLResponse` and
  `StreamingResponse`.
